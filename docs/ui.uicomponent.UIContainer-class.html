The constructor for UIComponent

Parameters:
  • parent - The parent component which contains this component. This should be an instance of UIComponent or any of its sub-classes. All components should have a parent with the exception of ui.screen.Screen. If you wish to draw a component directly to the screen, then its parent should be set to ui.screen.Screen. If a component is going to be added to a UIContainer using either UIContainer.addComponent or UIContainer.addComponents, then parent may be set to None, as either of those methods will set the parent attribute of the added components.
